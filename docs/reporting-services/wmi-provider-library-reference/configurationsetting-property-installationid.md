# <a name="configurationsetting-property---installationid"></a>ConfigurationSetting 属性 - InstallationID

## <a name="syntax"></a>语法  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
