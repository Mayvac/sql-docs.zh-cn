# <a name="sqlsetpos-visual-foxpro-odbc-driver"></a>SQLSetPos（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅 [ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：级别2  
  
 设置行集中的光标位置。 将游标定位到行集中的特定行后，可以使用 **SQLSetPos** 与 [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) 从未绑定的列中检索行。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLSetPos](../../odbc/reference/syntax/sqlsetpos-function.md) 。
