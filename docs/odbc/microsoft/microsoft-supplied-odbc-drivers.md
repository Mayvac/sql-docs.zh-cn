# <a name="microsoft-supplied-odbc-drivers"></a>Microsoft 提供的 ODBC 驱动程序
*驱动程序* 是在 ODBC API 中实现函数的库。 每个驱动程序特定于特定的数据库管理系统 (DBMS) 。 在大多数情况下，驱动程序将公开底层 DBMS 的功能;它们不需要实现 DBMS 不支持的功能。 驱动程序执行特定的任务，可以分为两个类别：基于文件和基于 DBMS。 有关驱动程序和驱动程序体系结构的详细信息，请参阅 [驱动程序](../../odbc/reference/drivers.md)。
