# <a name="encryption-access-driver"></a>加密（Access 驱动程序）
只有在版本4时，此控件才可用 (。*x* 格式已选择。 )   
  
 随着 Microsoft Jet 数据库引擎4.0 版本的发布，在创建访问数据库时允许数据库加密。 有关其他信息，请参阅 *Jet 数据库引擎程序员参考* 。
