# <a name="sqlgettypeinfo-paradox-driver"></a>SQLGetTypeInfo（Paradox 驱动程序）
> [!NOTE]  
>  本主题提供了特定于驱动程序的信息。 有关此函数的常规信息，请参阅 [ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 由 **SQLGetTypeInfo** 生成的表中返回的类型 (TYPE_NAME) 的名称将是数据源最常使用的名称。  
  
 对于 Byte、Counter、Double、Single、Long 和 Short 数据类型，将在可搜索列中返回 SQL_ALL_EXCEPT_LIKE。  (可以通过使用 ODBC 规范转换函数将值转换为字符，然后执行比较来实现类似的功能。 )
