# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅 [ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：核心级别  
  
 从结果集中检索一行，并将其放入之前对 [SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md)的调用指定的位置。 准备驱动程序，以便为未绑定的列调用 [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) 。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) 。
