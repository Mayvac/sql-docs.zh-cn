# <a name="conformance-levels"></a>一致性级别
ODBC 驱动程序允许应用程序访问不同的数据源。 每个驱动程序允许应用程序在运行时确定驱动程序和每个数据源支持的 ODBC 功能和 SQL 语法。 这不是设计为使用单个驱动程序或小型的已知驱动程序集的应用程序所必需的，因为这些应用程序只需写入该驱动程序或驱动程序的功能。 若要帮助应用程序发现驱动程序和数据源功能，可使用两个符合性方面： ODBC 接口和 SQL 语法。
