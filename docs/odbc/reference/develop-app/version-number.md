# <a name="version-number"></a>版本号
ODBC 有多种版本，每个版本都具有不同的功能。 应用程序通过使用 SQL_ODBC_VER 和 SQL_DRIVER_ODBC_VER 选项调用 **SQLGetInfo** 来确定驱动程序管理器和特定驱动程序支持的 ODBC 版本。
