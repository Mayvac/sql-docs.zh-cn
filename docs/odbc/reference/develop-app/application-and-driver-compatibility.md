# <a name="application-and-driver-compatibility"></a>应用程序和驱动程序兼容性
除了版本之外，ODBC 应用程序和驱动程序也属于多个类别。 其中一些应用程序与某些驱动程序不兼容;在其他情况下，应用程序或驱动程序的类型可能会影响它们之间的向后兼容性问题。  
  
 本部分包含以下主题。  
  
-   [应用程序的类型](../../../odbc/reference/develop-app/types-of-applications.md)  
  
-   [驱动程序类型](../../../odbc/reference/develop-app/types-of-drivers.md)  
  
-   [兼容性矩阵](../../../odbc/reference/develop-app/compatibility-matrix.md)
