# <a name="backup-checksum-default"></a>备份校验和默认值
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  使用备份校验和默认设置，启用或禁用备份和还原期间的备份校验和。  
  
 下表中列出了该选项的可能值：  
  
|||  
|-|-|  
|**值**|**含义**|  
|0|已禁用。 这是默认设置。|  
|1|已启用|  
  
 设置立即生效。
