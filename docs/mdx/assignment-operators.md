# <a name="assignment-operators"></a>赋值运算符


  在多维表达式 (MDX) 中，赋值运算符就是等号 (=)。  
  
 赋值运算符用于将值赋给 MDX 脚本中的子多维数据集。 有关详细信息，请参阅 [MDX 脚本编写基础知识 (Analysis Services)](/analysis-services/multidimensional-models/mdx/mdx-scripting-fundamentals-analysis-services)。
