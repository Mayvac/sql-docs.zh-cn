# <a name="-positive-mdx"></a>+（正值）(MDX)


  执行一元运算，以返回数值表达式的正值。  
  
## <a name="syntax"></a>语法  
  
```  
  
+ Numeric_Expression  
```  
  
#### <a name="parameters"></a>参数  
 *Numeric_Expression*  
 返回数值的有效多维表达式 (MDX) 表达式。  
  
## <a name="return-value"></a>返回值  
 一个具有指定参数的数据类型的正值。
