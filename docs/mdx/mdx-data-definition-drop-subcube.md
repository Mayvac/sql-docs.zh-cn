# <a name="mdx-data-definition---drop-subcube"></a>MDX 数据定义 - DROP SUBCUBE


  删除指定的子多维数据集，以恢复到以前定义的具有指定名称的多维数据集或子多维数据集定义。  
  
## <a name="syntax"></a>语法  
  
```  
  
DROP SUBCUBE Subcube_Name   
```  
  
## <a name="arguments"></a>参数  
 *Subcube_Name*  
 提供子多维数据集名称的有效字符串表达式。
