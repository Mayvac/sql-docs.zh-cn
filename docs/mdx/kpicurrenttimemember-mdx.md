# <a name="kpicurrenttimemember-mdx"></a>KPICurrentTimeMember (MDX)


  返回指定关键绩效指标 (KPI) 的当前时间成员。  
  
## <a name="syntax"></a>语法  
  
```  
  
KPICurrentTimeMember(KPI_Name)  
```  
  
## <a name="arguments"></a>参数  
 *KPI_Nam*e  
 指定 KPI 名称的有效字符串表达式。  
  
## <a name="remarks"></a>备注  
 KPI 可具有与时间维度的默认成员不同的时间成员。
