# <a name="query-with-joins-visual-database-tools"></a>使用联接进行查询 (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
查询结果可包含来自多个表或表值对象的数据。 若要组合来自多个表值对象的数据，可使用 SQL 的 JOIN 操作。  
  
有关使用多个表创建查询的信息，请参阅以下主题：  
  
## <a name="in-this-section"></a>本节内容  
[修改联接运算符 (Visual Database Tools)](../../ssms/visual-db-tools/modify-join-operators-visual-database-tools.md)  
指定应使用等号 (=) 之外的其他运算符联接表。  
  
[查询和视图设计器如何表示联接 (Visual Database Tools)](../../ssms/visual-db-tools/how-the-query-and-view-designer-represents-joins-visual-database-tools.md)  
解释联接在“关系图”窗格中的图形表示形式。  
  
[自动联接表 (Visual Database Tools)](../../ssms/visual-db-tools/join-tables-automatically-visual-database-tools.md)  
介绍设置由查询和视图设计器决定是否应联接表的步骤。  
  
[手动联接表 (Visual Database Tools)](../../ssms/visual-db-tools/join-tables-manually-visual-database-tools.md)  
介绍在“关系图”窗格中手动创建联接的步骤。  
  
[在多个列上联接表 (Visual Database Tools)](../../ssms/visual-db-tools/join-tables-on-multiple-columns-visual-database-tools.md)  
介绍按多个条件联接多个表的步骤。  
  
[创建外部联接 (Visual Database Tools)](../../ssms/visual-db-tools/create-outer-joins-visual-database-tools.md)  
指定即使某些行与相应表中的行不匹配，联接表也应包含这些行。  
  
[删除联接 (Visual Database Tools)](../../ssms/visual-db-tools/remove-joins-visual-database-tools.md)  
介绍移除表之间的联接的步骤。  
  
[自动创建自联接 (Visual Database Tools)](../../ssms/visual-db-tools/create-self-joins-automatically-visual-database-tools.md)  
介绍设置由查询和视图设计器创建自联接的步骤。  
  
[手动创建自联接 (Visual Database Tools)](../../ssms/visual-db-tools/create-self-joins-manually-visual-database-tools.md)  
介绍使用联接在单个表中查找数据子集的步骤。  
  
[查看联接属性 (Visual Database Tools)](../../ssms/visual-db-tools/view-join-properties-visual-database-tools.md)  
介绍查看联接的属性的步骤。
