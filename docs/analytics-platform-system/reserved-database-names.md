# <a name="reserved-database-names-in-parallel-data-warehouse"></a>并行数据仓库中的保留数据库名称
保留的数据库名称是无法在 SQL Server PDW 中分配给用户数据库的名称。  
  
保留数据库名称列表：  
  
-   主  
  
-   模型  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
