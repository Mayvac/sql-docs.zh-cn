# <a name="command-property-adox"></a>Command 属性 (ADOX)
指定可用于创建或执行该过程的 ADO [命令](../../../ado/reference/ado-api/command-object-ado.md) 对象。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回有效的 ADO **命令** 对象。  
  
## <a name="remarks"></a>备注  
 如果提供程序不支持保留命令，则在获取和设置此属性时将发生错误。
