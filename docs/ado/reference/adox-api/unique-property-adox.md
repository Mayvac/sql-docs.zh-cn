# <a name="unique-property-adox"></a>Unique 属性 (ADOX)
指示索引键是否必须唯一。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回一个 **Boolean** 值。  
  
## <a name="remarks"></a>备注  
 默认值为 **False**。  
  
 对于已追加到集合的 [索引](../../../ado/reference/adox-api/index-object-adox.md) 对象，此属性是只读的。
