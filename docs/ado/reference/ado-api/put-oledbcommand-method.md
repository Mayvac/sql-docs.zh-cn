# <a name="put_oledbcommand-method"></a>put_OLEDBCommand 方法
此方法不执行任何操作，并且始终返回 S_OK。  
  
## <a name="syntax"></a>语法  
  
```  
  
HRESULT put_OLEDBCommand(  
      IUnknown *pOLEDBCommand  
);  
```  
  
#### <a name="parameters"></a>参数  
 *pOLEDBCommand*  
 中指向 OLE DB Command 对象的指针。
