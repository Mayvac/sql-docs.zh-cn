# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
指定要从 [记录集中](./recordset-object-ado.md)检索的记录数。  
  
|返回的常量|Value|说明|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|从当前位置或由[GetRows](./getrows-method-ado.md)方法的*Start*参数指定的书签检索记录**集中**的其余记录。|
