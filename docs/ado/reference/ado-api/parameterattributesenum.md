# <a name="parameterattributesenum"></a>ParameterAttributesEnum
指定 [参数](../../../ado/reference/ado-api/parameter-object.md) 对象的特性。  
  
|返回的常量|值|描述|  
|--------------|-----------|-----------------|  
|**adParamSigned**|16|指示参数接受有符号的值。|  
|**adParamNullable**|64|指示参数接受 null 值。|  
|**adParamLong**|128|指示参数接受长二进制数据。|
