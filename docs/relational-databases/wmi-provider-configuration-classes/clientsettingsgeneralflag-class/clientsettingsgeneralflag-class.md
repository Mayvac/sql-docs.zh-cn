# <a name="clientsettingsgeneralflag-class"></a>ClientSettingsGeneralFlag 类
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  [ClientSettingsGeneralFlag 类](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md)表示 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] 客户端实例的常规标志设置 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
 [ClientSettingsGeneralFlag 类](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md)可用于执行以下操作：  
  
-   查看标志名称。  
  
-   查看标志索引值。  
  
-   查看并修改标志值。
