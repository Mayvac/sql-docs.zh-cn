# <a name="user-and-group-permissions-functional-area-master-data-manager"></a>用户和组权限功能区域（主数据管理器）
  在“用户和组权限”功能区域中，管理员可以向功能区域、属性（在“模型”选项卡上）和成员（在“层次结构成员”选项卡上）授予权限。 解决重叠的权限来确定用户对每个单独属性值的权限。
