# <a name="step-3-testing-the-lesson-6-package"></a>步骤 3：测试 Lesson 6 包
  在运行时，包从 VarFolderName 参数获取 Directory 属性的值。  
  
 若要验证该包在运行时是否使用新值更新了 Directory 属性，只需执行该包。 由于只向新目录中复制了三个示例数据文件，因此该数据流将只运行三次，而不遍历原始文件夹中的 14 个文件。
