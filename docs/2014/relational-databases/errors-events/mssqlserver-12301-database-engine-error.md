# <a name="mssqlserver_12301"></a>MSSQLSERVER_12301
    
## <a name="details"></a>详细信息  
  
|Attribute|值|  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|12301|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|HK_UNSUPPORTED_NULLABLE_COLUMNS|  
|消息正文|“*construct*”不支持索引键中具有可以为 Null 的列。|  
  
## <a name="user-action"></a>用户操作  
 请勿在索引键中使用可以为 Null 的列。
