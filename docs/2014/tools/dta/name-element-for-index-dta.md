# <a name="name-element-for-index-dta"></a>索引的名称元素 (DTA)
  在用户指定的配置中为索引指定一个名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
<Create>  
    <Index>  
        <Name>...</Name>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|说明|  
|--------------------|-----------------|  
|**数据类型和长度**|`string`，长度没有限制。|  
|**默认值**|无。|  
|**出现次数**|对于每个 `Index` 元素必须使用一次。|  
  
## <a name="element-relationships"></a>元素关系  
  
|关系|元素|  
|------------------|--------------|  
|**父元素**|[索引元素 (DTA)](index-element-dta.md)|  
|**子元素**|无。|
