# <a name="mdx-builder-analysis-services---multidimensional-data"></a>MDX 生成器（Analysis Services -多维数据）
  可以使用 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] 或 [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 中的“MDX 生成器”**** 对话框生成多维表达式 (MDX) 表达式。 通过单击 "**允许读取多维数据集内容**" 选项的 "**编辑 mdx**省略号" 按钮（**...**），可以显示 " **Mdx 生成器**" 对话框，在**角色设计器**的 "**单元数据**" 页上，单击 "**允许读取**单元格内容的显示内容" 选项或 "**允许读取和写入多维数据集内容**" 选项。  
  
## <a name="options"></a>选项  
  
|术语|定义|  
|----------|----------------|  
|**表达式**|键入要使用的 MDX 表达式。|  
|**勾选标记**|单击 **“检查”** 可以测试 **“表达式”** 中定义的 MDX 表达式。|  
|**元数据**|显示可以包含在 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] “表达式” **中所定义的 MDX 表达式中的当前**对象的元数据。<br /><br /> 通过右键单击所选项，然后从上下文菜单中选择“复制”****，或者将所选项拖到“表达式”****，可以复制所选项的 MDX 语法。|  
|**函数**|显示当前 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 实例的可用 MDX 函数。 可以从 MDSCHEMA_FUNCTIONS 架构行集中检索所列项。<br /><br /> 通过右键单击所选项，然后从上下文菜单中选择“复制”****，或者将所选项拖到“表达式”****，可以复制所选项的 MDX 语法。|
