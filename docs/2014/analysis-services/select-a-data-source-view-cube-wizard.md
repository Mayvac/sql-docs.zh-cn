# <a name="select-a-data-source-view-cube-wizard"></a>选择数据源视图（多维数据集向导）
  可以使用 **“选择数据源视图”** 页，为空多维数据集选择数据源视图。  
  
> [!NOTE]  
>   仅当在 **“选择创建方法”** 页上选择了 **“创建空的多维数据集”** 之后，才会显示此页。  
  
## <a name="options"></a>选项  
 **数据源视图**  
 选择要用于空的多维数据集的数据源视图。  
  
 **无**  
 如果不想使用数据源视图，则请选择此选项。 如果要稍后生成数据源视图，或者计划只将多维数据集和链接度量值组一起使用，则可以使用此选项。
